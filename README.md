# Data Storytelling and Statistical Validation: IPL (2008–2024)

## 📌 Project Overview
Venue impact, and toss influence across IPL seasons from 2008 to 2024. The analysis combines visualization techniques with hypothesis testing to support data-driven cricket strategies. 

## 🎯 Objectives

* Analyze team and player performances across IPL seasons.
* Identify venue-specific trends and match patterns.
* Evaluate the influence of toss outcomes on match results.
* Validate insights using statistical hypothesis testing.
* Present findings through interactive dashboards and visual storytelling. 

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, SciPy)
* Jupyter Notebook
* Power BI
* GitHub
* Gamma AI

These tools were used for data cleaning, visualization, statistical analysis, and project documentation. 

## 📊 Key Insights

* Mumbai Indians emerged as the most successful IPL franchise based on long-term performance and consistency.
* Chris Gayle recorded the highest aggregate runs, while Yuzvendra Chahal became the leading wicket-taker.
* Eden Gardens hosted the highest number of IPL matches, highlighting its significance as a major venue.
* Toss winners won slightly more matches, indicating a potential relationship between toss outcomes and match results. 

## 📈 Statistical Validation

A Chi-Square Test of Independence was performed to examine the relationship between toss results and match outcomes. The results rejected the null hypothesis, indicating a statistically significant association between winning the toss and winning the match. This validation strengthens the reliability of the analytical findings. 

## 🚀 Conclusion

This project demonstrates how data storytelling and statistical analysis can be combined to generate actionable insights in sports analytics. By integrating interactive dashboards with hypothesis testing, the analysis provides valuable recommendations for team strategy, player evaluation, and match planning.
